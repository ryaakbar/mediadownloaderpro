FAQ
Kenapa hasilnya error / media tidak ditemukan?

Pastikan URL yang di-paste valid dan merupakan link langsung ke video/foto. Link dari Stories atau konten private tidak bisa didownload.

Apakah ada limit download?

Saat ini tidak ada limit resmi. Namun untuk menjaga kestabilan server, hindari spam request dalam waktu singkat.

Kenapa video YouTube tidak bisa HD?

Kualitas yang tersedia bergantung dari API yang digunakan. Untuk YouTube, kualitas maksimal yang tersedia mungkin terbatas.

Apakah aman untuk digunakan?

Tool ini tidak menyimpan data URL atau media lo di server. Semua proses dilakukan secara real-time dan tidak ada logging.
